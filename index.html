Lekce 1

Jak počítač čte náš kód?

  • 1. Strojový kód: Procesor zná jen 0 a 1 (např. 11010010). Zcela nečitelné.
  • 2. Assembler (ASM): Instrukce dostaly zkratky (např. ADD, LDA). Pořád složité a závislé na typu procesoru.
  • 3. Kompilované jazyky (C, C++): Píšeme kód podobný angličtině. Překladač ho najednou celý přeloží do 0 a 1. Velmi rychlé, ale nepřenositelné mezi systémy.
  • 4. Virtuální stroj (C#, Java): Kód se přeloží do "mezikódu". Až při spuštění programu se mezikód překládá pro konkrétní počítač.

Rozdíl: C# vs. .NET


C# (C-Sharp)

Je programovací jazyk.
Určuje slovník a gramatiku, kterou budeme psát.

.NET

Je platforma (motor).
Poskytuje nám hotové knihovny a stará se o běh našeho kódu (tzv. CLR).

Python vs. C#

Zatímco Python je volný, C# má rád pořádek.

Python


                        # Nepíšeme datové typy
                        jmeno = "David"
                        
                        # Odsazení dělá blok
                        if jmeno == "David":
                            print("Ahoj!")

C#


                        // Musíme říct, že jde o text (string)
                        string jmeno = "David";
                        
                        // Bloky dělají závorky {}
                        if (jmeno == "David")
                        {
                            Console.WriteLine("Ahoj!");
                        } // A pozor na středníky!

Anatomie programu

Když založíme nový projekt, uvidíme toto:


                using System; // Připojí základní nástroje (např. Console)

                namespace MojeAplikace // Název našeho projektu
                {
                    class Program // Hlavní krabice
                    {
                        static void Main(string[] args) // ZDE PROGRAM ZAČÍNÁ!
                        {
                            Console.WriteLine("Ahoj světe!");
                        }
                    }
                }

(Poznámka: Nové verze .NETu umí tuto "omáčku" schovat, ale pod kapotou tam stále je).

💻 ÚKOL 1: První spuštění

  1. Otevřete Visual Studio.
  2. Vytvořte Konzolovou aplikaci (Console App).
  3. Pojmenujte ji např. Lekce1.
  4. Dovnitř metody Main napište kód, který do konzole vypíše:
    "Vítej v kurzu C#!"
  5. Spusťte program (zelené tlačítko Play nebo F5).

Proměnné (Šuplíky v paměti)

V C# musíme na každý šuplík nalepit štítek, co v něm bude (Datový typ). Pokud tam dáme něco jiného, program spadne.

Typ K čemu slouží Příklad
int Celá čísla int vek = 16;
string Text (do uvozovek) string jmeno = "Karel";
double Desetinná čísla (s tečkou) double plat = 150.5;
bool Pravda / Nepravda bool zije = true;

Výpis proměnných

Když chceme do textu vložit proměnnou, použijeme znak $ (tzv. interpolace textu).


                int pocetZamestnancu = 10;
                string sef = "David";

                // Znak $ nám dovolí používat složené závorky v textu
                Console.WriteLine($"Firmu vede {sef} a má {pocetZamestnancu} lidí.");

💻 ÚKOL 2: Proměnné v praxi

  1. Založte si proměnnou typu string a vložte do ní jméno vaší oblíbené postavy/osoby.
  2. Založte si proměnnou typu int a vložte do ní její věk.
  3. Pomocí Console.WriteLine a znaku $ vypište větu:
    "Jmenuji se [Jméno] a je mi [Věk] let."

Vstup od uživatele

Program by měl umět reagovat. K přečtení toho, co uživatel napíše na klávesnici, slouží Console.ReadLine().


                Console.WriteLine("Jak se jmenuješ?");
                
                // Program se zastaví a čeká na stisk ENTER
                string zadaneJmeno = Console.ReadLine(); 
                
                Console.WriteLine($"Ahoj {zadaneJmeno}!");

⚠️ Zrádná čísla (Parsing)

Console.ReadLine() čte VŽDY pouze text (string).

Text "20" a číslo 20 jsou pro počítač dvě různé věci!


                Console.WriteLine("Zadej svůj věk:");
                
                // Tímto získáme text (např. "20")
                string vstupText = Console.ReadLine();
                
                // Překlad textu na matematické číslo!
                int vekCislo = int.Parse(vstupText);
                
                // Nyní můžeme s číslem počítat
                int zbyvaDoDuchodu = 65 - vekCislo;
                Console.WriteLine($"Do důchodu ti zbývá {zbyvaDoDuchodu} let.");

  • Zeptejte se na Pracovní pozici.
  • Zeptejte se na Věk (nezapomeňte převést text na číslo pomocí int.Parse).
  • Zeptejte se na měsíční Plat (opět převést).
  • Program následně vynásobí plat 12 (roční plat) a vypíše hezky naformátovanou vizitku.

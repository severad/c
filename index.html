Vývoj v C# - Základy

1. Cíl kurzu a moderní vývoj

Cílem tohoto pololetí je osvojit si základy jazyka C# a platformy .NET. Namísto izolovaných teoretických úloh budeme postupně budovat komplexní konzolový informační systém (např. systém pro evidenci zaměstnanců nebo skladové hospodářství). Naučíte se navrhovat architekturu aplikace a pracovat s daty.

Role umělé inteligence (AI) ve vývoji

V současné praxi jsou nástroje jako GitHub Copilot nebo ChatGPT běžnou součástí práce programátora. Je však klíčové si uvědomit:

  • AI negeneruje myšlenky, ale kód. Vyřeší za vás syntaxi (jak se píše cyklus), ale nevyřeší za vás logiku a architekturu (kam ten cyklus umístit a jaká data má zpracovávat).
  • Odpovědnost nese vývojář. Vygenerovaný kód obsahuje často chyby nebo neodpovídá kontextu vaší aplikace. Bez hluboké znalosti jazyka a schopnosti kód číst nebudete schopni tyto chyby odhalit a opravit.
  • Naším cílem je naučit se C# na takové úrovni, abyste AI dokázali využívat jako nástroj, který řídíte a kontrolujete vy.

2. Z Pythonu do světa C#

Zatímco Python je dynamicky typovaný jazyk (typ proměnné se určuje za běhu), C# je staticky typovaný. To znamená, že musíte vždy přesně deklarovat, jaký typ dat bude proměnná obsahovat. To předchází mnoha chybám ve velkých projektech.

Python (Dynamický)
# Proměnná může obsahovat cokoliv
jmeno = "Martin"
vek = 20

print(f"Jmenuje se {jmeno} a je mu {vek}")
C# (Statický)
// Musíme definovat datový typ (string, int)
string jmeno = "Martin";
int vek = 20;

Console.WriteLine($"Jmenuje se {jmeno} a je mu {vek}");

Základní pravidlo C#: Každý příkaz musí být ukončen středníkem ;

3. Vývojové prostředí: Visual Studio

Pro vývoj v C# budeme využívat IDE (Integrované vývojové prostředí) Visual Studio, které poskytuje pokročilé nástroje pro analýzu kódu a debugging (hledání chyb).

  1. Otevřete Visual Studio.
  2. Zvolte Vytvořit nový projekt (Create a new project).
  3. Vyberte šablonu Konzolová aplikace (Console App) pro C#.
  4. Projekt vhodně pojmenujte (např. FiremniSystem).
  5. Při volbě frameworku ponechte výchozí nastavení (využijeme moderní přístup tzv. Top-level statements, díky kterému náš kód připomíná Python skripty).

4. Proměnné a základní výpis

Program.cs 
// Deklarace a inicializace proměnných
int age = 20;
string name = "Martin";

// Základní výpis do konzole
Console.WriteLine(age);
Console.WriteLine(name);

// Interpolace textu (vložení proměnných přímo do řetězce pomocí znaku $)
Console.WriteLine($"Jmenuje se {name} a je mu {age} let.");

Základní datové typy, které budeme potřebovat:

  • int
    Celá čísla (věk, počet kusů na skladě)
  • string
    Textové řetězce (jméno, název produktu)
  • double / decimal
    Desetinná čísla (mzda, cena položky)
  • bool
    Logická hodnota true/false (je aktivní?, má přístup?)

5. Uživatelský vstup a konverze (Parsování)

Informační systém musí umět přijímat data od uživatele. K tomu slouží metoda Console.ReadLine(). Je zde ale jeden zásadní koncept:

Pozor na text vs. číslo

Vše, co uživatel napíše do konzole, C# vnímá jako text (string), i když zadá "25". Pokud s tímto vstupem chceme počítat, musíme ho překonvertovat na číslo pomocí int.Parse(). 

// 1. Získání textového vstupu
Console.Write("Zadejte jméno zaměstnance: "); // Write neprovede odřádkování
string zamestnanec = Console.ReadLine();

// 2. Získání číselného vstupu vyžaduje konverzi
Console.Write("Zadejte věk zaměstnance: ");
string vstupVek = Console.ReadLine(); // Načteme text, např. "30"
int vekZamestnance = int.Parse(vstupVek); // Převedeme text "30" na číslo 30

// Nyní můžeme provádět matematické operace
int rokNarozeni = 2024 - vekZamestnance;

Console.WriteLine($"Zaměstnanec {zamestnanec} byl úspěšně zaevidován. Odhadovaný rok narození: {rokNarozeni}.");

Samostatná práce: Evidenční karta zaměstnance

Rozšiřte náš základní program a vytvořte registrační modul pro nový firemní systém.

Požadavky zadání:

  • Program vypíše uvítací hlavičku firemního systému.
  • Program si postupně vyžádá vstupy: Jméno a příjmení, Název pozice a Nástupní plat (v celých číslech).
  • Plat musí být korektně převeden na číselný typ (int).
  • Program vypočítá odhadovanou roční mzdu (nástupní plat vynásobený 12).
  • Na závěr program naformátuje a přehledně vypíše zadané údaje do podoby "Karty zaměstnance" na obrazovku.

// Očekávaný výstup v konzoli:

=== HR SYSTÉM v1.0 ===
Zadejte jméno zaměstnance: Jan Novák
Zadejte pracovní pozici: IT Analytik
Zadejte měsíční mzdu (Kč): 45000

--- KARTA ZAMĚSTNANCE ---
Jméno: Jan Novák
Pozice: IT Analytik
Měsíční mzda: 45000 Kč
Odhadovaný roční náklad: 540000 Kč
-------------------------
Záznam úspěšně uložen do paměti.

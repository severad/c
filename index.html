Vývoj v C# - Cesta do Arény

1. Proč jsme tady? (Aneb proč nás AI ještě nenahradila)

Naším cílem pro toto pololetí není biflovat se poučky. Naším cílem je naprogramovat plně funkční textovou RPG hru (Arénu), kde proti sobě budou bojovat Rytíři, Skřeti a Mágové s různými statistikami a schopnostmi.

Slon v místnosti: Umělá inteligence

Možná si říkáte: "Proč se to učit, když mi kód napíše ChatGPT nebo GitHub Copilot?"

  • AI je jako velmi rychlý asistent, který je ale občas úplně mimo. Vygeneruje vám kód, ale když v něm bude chyba (a ona tam bude), nebo nebude pasovat do vaší hry, bez znalosti základů budete naprosto ztraceni.
  • Myšlení se vygenerovat nedá. Programování není o psaní textu. Je to o schopnosti rozbít velký problém (chci vytvořit hru) na sérii malých, logických kroků. Tuto schopnost využijete v jakémkoliv oboru.
  • Naším cílem je naučit se jazyk C# tak dobře, abyste mohli AI používat jako profíci – jako nástroj, který řídíte vy, ne on vás.

2. Z Pythonu do světa C#

Python už znáte. Byl skvělý na začátek, protože nevyžadoval moc příprav. C# je robustnější, profesionální jazyk, na kterém běží velká část bank, firemních systémů, ale i hry (např. engine Unity).

🐍 Python (Dynamický a volný)
# Nemusím říkat, jaký typ to je
jmeno = "Ignác"
vek = 120

# Odsazení určuje blok kódu
if vek > 100:
    print("To je ale starý Rytíř!")
💠 C# (Statický a striktní)
// Musím přesně říct, co ukládám! (string, int)
string jmeno = "Ignác";
int vek = 120;

// Bloky určují chlupaté závorky {} a středníky!
if (vek > 100) 
{
    Console.WriteLine("To je ale starý Rytíř!");
}

⚠️ Zlaté pravidlo C#: Na konci (téměř) každého řádku MUST být středník ;

3. Naše zbrojovka: Visual Studio

Nebudeme psát kód v Notepadu. Použijeme Visual Studio (pozor, neplést s Visual Studio Code). Je to IDE (Integrované vývojové prostředí).

  1. Otevřete Visual Studio.
  2. Klikněte na Vytvořit nový projekt (Create a new project).
  3. Do vyhledávání nahoře napište Konzolová aplikace (nebo Console App). Vyberte tu s logem C#.
  4. Pojmenujte projekt např. MojePrvniArena.
  5. Nechte zaškrtnuté ".NET 8.0" (nebo nejnovější) a odklikněte "Do not use top-level statements" (Nepoužívat příkazy nejvyšší úrovně) - k tomu se hned dostaneme.

4. Anatomie programu v C#

C# vás nutí mít ve věcech pořádek od prvního řádku. Každý program musí mít svou obálku.

Program.cs 
using System; // Připojí základní knihovny (abychom mohli psát do konzole)

namespace MojePrvniArena // Název složky/prostoru pro náš kód
{
    class Program // Třída - hlavní krabice programu
    {
        // Metoda Main - TADY PROGRAM ZAČÍNÁ! Sem budeme zatím psát všechen kód.
        static void Main(string[] args)
        {
            Console.WriteLine("Vítej v Aréně, hrdino!"); // Vypíše text a odřádkuje
            
            // Zabrání zavření konzole, dokud nezmáčkneme Enter
            Console.ReadLine(); 
        }
    }
}

Jak spustit program? Nahoře ve Visual Studiu klikněte na zelené tlačítko ▶️ Play (nebo klávesa F5).

5. Tvorba postavy (Proměnné a Vstupy)

Abychom mohli hrát, musíme si ukládat data (zdraví, jméno). V C# se používají typy proměnných:

  • int - celá čísla (životy, síla).
    Př: int zivoty = 100;
  • string - text (jméno).
    Př: string jmeno = "Belzebub";
  • double - desetinná čísla (zlato).
    Př: double zlato = 15.5;
  • bool - pravda/nepravda (žije?).
    Př: bool jeZivy = true;

Komunikace s hráčem

Console.Write("Zadej jméno svého hrdiny: "); // Výpis bez odřádkování
string jmenoHrdiny = Console.ReadLine(); // Přečte text z klávesnice, dokud hráč nedá Enter

// Spojování textu a proměnné (dolar $ před uvozovkami dělá zázraky!)
Console.WriteLine($"Vítej v aréně, {jmenoHrdiny}! Tvá cesta začíná.");

Pasti pro začátečníky: Čísla z klávesnice

Console.ReadLine() VŽDY vrací text (string). Pokud chcete po uživateli číslo, musíte text přeložit na číslo (tzv. parsování). 

Console.Write("Zadej sílu (1-10): ");
string vstupSila = Console.ReadLine();
int sila = int.Parse(vstupSila); // Přemění text "5" na matematické číslo 5

Samostatná Mise: Generátor postavy

Vaším prvním úkolem je vytvořit aplikaci, která uvítá hráče, zeptá se ho na vlastnosti postavy a pak vypíše krásný "Průkaz hrdiny".

Požadavky:

  • Program pozdraví a vyzve hráče k vytvoření postavy.
  • Zeptá se na: Jméno (text).
  • Zeptá se na: Povolání (text - např. Rytíř, Skřet, Mág).
  • Zeptá se na: Věk (číslo - nezapomeňte na int.Parse).
  • Nakonec vypíše hezky naformátovanou tabulku (nebo rámeček pomocí znaků = a |) se všemi údaji.
  • Bonus pro rychlíky: Spojte věk s nějakou matematikou (např. vypište: "Do důchodu ti zbývá " + (65 - vek) + " let bojů v Aréně!").

Příklad očekávaného výstupu v konzoli:

Vítej v generátoru postav!
Zadej jméno hrdiny: Ignác
Zadej povolání: Rytíř
Zadej věk: 35

============================
PRŮKAZ HRDINY
============================
Jméno: Ignác
Povolání: Rytíř
Věk: 35
Do důchodu ti zbývá: 30 let.
